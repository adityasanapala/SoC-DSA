# SoC-DSA

Welcome to the completely revamped **Seasons of Code 2026** repository! This repository contains the weekly tasks, curated resources, and learning materials for the **Data Structures, Algorithms, and Competitive Programming (DSA & CP)** track under the 2026 edition of **Seasons of Code**.

This project is designed to help you build a strong foundation in **Data Structures & Algorithms (DSA)** and **Competitive Programming**, while also giving you the opportunity to work on hands-on projects for deeper exploration and practical experience.
